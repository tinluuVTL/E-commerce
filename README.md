## Bắt đầu

Đầu tiên, chạy máy chủ phát triển:

```bash
npm install
# hoặc
yarn instll

# và Bây giờ:

npm run dev
# hoặc
yarn dev
```
Mở [http://localhost:5173/](http://localhost:5173/) bằng trình duyệt của bạn để xem kết quả.
